# Proyecto-de-titulacion-licenciatura
Archivos de entrada, datos, graficos y documentación soporte del proyecto de investigación de la carrera de Ingeniería Física en las materias de ASE II y ASE III de la Universidad Iberoamericana.
En este trabajo presentamos los resultados que obtuvimos al analizar cómo se van modificando las trayectorias de asteroides potencialmente peligrosos con relación a la Tierra y el riesgo de colisión que representan para nuestro planeta en los próximos mil años. Además, investigamos si estas poblaciones son estables utilizando simulaciones numéricas de partículas generadas aleatoriamente, tomando como referencia la población de asteroides observada en la actualidad, mediante el uso del código MERCURY 6.2. Como objetivos adicionales, se buscó dominar los principios de la mecánica celeste, así como el manejo y la aplicación exitosa de ciertos lenguajes de programación necesarios para estudiar estos cuerpos.
